Programming Games

Cigarmonica - Game Design Club Group Project (Link)- 6 downloads

  • For this game, I worked with the rest of the Game Design Club at my college
  • We all worked together to design the game, I then helped write some of the NPC dialogue (particularly level 3), programmed many of the game's systems, and implemented the game into Unity
  • We designed a simple gameplay loop of players being given an item (normally in the previous level), and having to figure out whom to give it to by interacting with the NPCs
  • Only by giving the correct NPC their corresponding item, could players be able to progress, eventually find out what happens to Llewyllyn, and escape the prison

Games I made during Game Jams - MicroJam Game Submissions (Link)

  • For these games, I worked by myself as I was unable to find people I felt comfortable working with.
  • I had 2 days for each of these games to make a functioning video game that met the theme (which I knew before the jam started) and the prerequisite (which was revealed at the start of the jam), both of which would impact how I made the games.
  • I used these game jams as opportunities to get better at using various programming patterns I had recently learned about (like Singletons, Observers, and Composition), in addition to me challenging myself at times to figure out how game developers make things like notes in video games.
  • Although none of these games ever won, I noticed a massive growth in the code I wrote, as it changed from being heavily coupled code during my first game jam, to being largely modulable.
  • I also gained a better understanding of how long it takes for me to make things (specifically when I have pressure outside myself being applied), and as a result, my games were able to become more polished within largely the same amounts of time required

Web Development

Reading Ruler Firefox Plugin - Personal Project (Link)

  • After struggling to read what I felt was a particularly poorly formatted article for a class, I sought to find a reading ruler plug in, but couldn't find any (I didn't know they were called add-ons, and not extensions, on firefox)
  • I decided to fix this problem by making myself a reading ruler plug in for Firefox using HTML, CSS, and JavaScript.
  • I figured out how to make a reading ruler that obscured all text outside an adjustable area.
  • This project took me about a month to get working, with a several month gap between first getting the ruler to work, and actually publishing it, as to be honest, I kind of forgot I'd been working on it as I was using all my spare time on it; spare time towards the end of the month that I rapidly started to lose.
